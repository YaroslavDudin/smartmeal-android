# smartmeal-android
# 🥗 Умный ужин (SmartMeal)

[cite_start]Мобильное приложение, которое избавляет от ежедневного вопроса «Что приготовить?»[cite: 435, 436]. [cite_start]Оно составляет персонализированное меню, подбирает рецепты и автоматически генерирует умный список продуктов для похода в супермаркет или заказа онлайн[cite: 436].

## 🎯 Концепция и Дизайн
* **Стиль:** Чисто, аппетитно, минимализм. [cite_start]Много воздуха[cite: 501].
* [cite_start]**Цветовая палитра:** «Съедобные» цвета — зеленый (свежесть), оранжевый (аппетит), белый[cite: 502].
* [cite_start]**Главное правило:** От открытия приложения до готового списка покупок — менее 5 кликов[cite: 503].

## 🛠 Технологический стек
Проект построен на современной клиент-серверной архитектуре:

* **Frontend (Мобильное приложение):**
  * [cite_start]Язык: Kotlin[cite: 472].
  * UI-фреймворк: Jetpack Compose.
  * Архитектура: Single Activity, Material 3.
* **Backend (Сервер и API):**
  * [cite_start]Язык: Python[cite: 473].
  * [cite_start]Фреймворк: Django & Django REST Framework[cite: 473].
* **База данных:**
  * [cite_start]СУБД: PostgreSQL[cite: 474].

## 🚀 Как запустить проект локально

### Запуск Backend (Django)
1. Откройте терминал и перейдите в папку сервера: `cd server`
2. Создайте виртуальное окружение: 
   * Windows: `python -m venv venv`
   * Mac/Linux: `python3 -m venv venv`
3. Активируйте окружение:
   * Windows: `venv\Scripts\activate`
   * Mac/Linux: `source venv/bin/activate`
4. Установите зависимости: `pip install -r requirements.txt`
5. Запустите локальный сервер: `python manage.py runserver`
6. Сервер будет доступен по адресу: `http://127.0.0.1:8000/`

### Запуск Frontend (Android)
1. Откройте папку `client/SmartMeal` в VS Code (или Android Studio).
2. Дождитесь автоматической синхронизации Gradle (или нажмите Sync).
3. Запустите проект на подключенном физическом устройстве или эмуляторе.

## 👥 Командная работа
* Не коммитим папку `venv/`, `__pycache__/`, локальную БД `db.sqlite3` и секретные ключи (`.env`).
* Файлы миграций базы данных (`server/meals/migrations/`) **обязательно** добавляем в коммиты!
* Ветки создаем под каждую новую фичу (например: `feature/menu-generator`, `fix/login-screen`).
